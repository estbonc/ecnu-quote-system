# **报价系统** #

## 报价系统是什么 ##

报价系统是居然网络设计工具配套报价引擎，支持基于装饰公司报价逻辑的报价系统。
## 报价系统功能模块 ##
报价系统功能模块主要有两大块，一块是报价系统管理平台，集成在运营管理后台。运营人员通过管理平台管理报价所需基础数据。另外一块是报价逻辑，报价引擎基于报价基数数据以及3D设计导出设计方案报价。
